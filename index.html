NAME

daemon — run detached from the controlling terminal

SYNOPSIS

daemon [-cf] [-g group] [-p pidfile] [-u user] command arguments ...

DESCRIPTION

The daemon utility detaches itself from the controlling terminal and executes the program specified by its arguments. Privileges may be lowered to the specified user and/or group.

The options are as follows:

-c
Change the current working directory to the root.
-f
Redirect standard input, standard output and standard error to /dev/null.
-g group
Drop privileges to specified group.
-p file
Write the PID of created process into the file.
-u user
Drop privileges to specified user.

DIAGNOSTICS

If the command cannot be executed, an error message is displayed on standard error unless the -f option is specified.
